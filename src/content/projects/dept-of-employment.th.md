## ภาพรวมโครงการ

กรมการจัดหางาน กระทรวงแรงงาน มอบหมายให้ บริษัท ทีซีเอ็ม เทคโนโลยี จำกัด ดำเนินการออกแบบ จัดหา และติดตั้งโครงสร้างพื้นฐานด้านไอทีสำหรับสำนักงานกลาง เพื่อรองรับระบบงานจัดหางานและการให้บริการประชาชนในรูปแบบดิจิทัลที่ขยายตัวอย่างต่อเนื่อง

## ขอบเขตงาน

- ออกแบบสถาปัตยกรรมระบบเครือข่ายและเซิร์ฟเวอร์สำหรับส่วนกลาง
- ติดตั้งและตั้งค่า Server และ Storage พร้อมระบบ Virtualization
- วางระบบ LAN ภายในอาคาร พร้อม Structured Cabling มาตรฐาน
- ติดตั้ง Active Directory และระบบบริหารจัดการผู้ใช้ (IAM)
- จัดทำระบบ Backup และ Disaster Recovery เบื้องต้น
- ทดสอบและส่งมอบระบบพร้อมเอกสารครบถ้วน
- ให้บริการ After-Sales Support หลังส่งมอบ

## ผลลัพธ์

ระบบไอทีที่ปรับปรุงใหม่รองรับการทำงานพร้อมกันของเจ้าหน้าที่จำนวนมาก รองรับการเพิ่มขึ้นของผู้ใช้บริการออนไลน์ และลดระยะเวลาในการประมวลผลคำขอรับบริการของประชาชน

## เทคโนโลยีที่ใช้

- Server และ Storage ระดับองค์กร
- ระบบ Virtualization
- Active Directory / IAM
- Structured Cabling และ LAN
- ระบบ Backup และ Disaster Recovery
