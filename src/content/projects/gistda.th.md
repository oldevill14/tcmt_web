## ภาพรวมโครงการ

สำนักงานพัฒนาเทคโนโลยีอวกาศและภูมิสารสนเทศ (GISTDA) เป็นหน่วยงานด้านเทคโนโลยีขั้นสูง ที่ต้องการโครงสร้างพื้นฐานไอทีที่มีความน่าเชื่อถือและประสิทธิภาพสูง บริษัท ทีซีเอ็ม เทคโนโลยี จำกัด ได้รับมอบหมายให้ออกแบบและปรับปรุงระบบเครือข่ายและโครงสร้างพื้นฐานเซิร์ฟเวอร์ให้รองรับปริมาณข้อมูลขนาดใหญ่จากงานด้านภูมิสารสนเทศ

## ขอบเขตงาน

- ออกแบบสถาปัตยกรรมเครือข่ายประสิทธิภาพสูง (High-performance Network)
- ติดตั้งอุปกรณ์เครือข่าย Core และ Distribution Layer ใหม่
- วางระบบ Structured Cabling ภายในอาคาร
- จัดหาและติดตั้ง Server และ Storage สำหรับรองรับข้อมูลขนาดใหญ่
- ตั้งค่าระบบ Firewall และ Access Control
- ทดสอบประสิทธิภาพและความเสถียรของระบบก่อนส่งมอบ
- จัดทำเอกสาร Network Diagram และคู่มือการใช้งาน

## ผลลัพธ์

โครงสร้างพื้นฐานไอทีที่ปรับปรุงใหม่รองรับปริมาณข้อมูลขนาดใหญ่จากระบบภูมิสารสนเทศและดาวเทียม พร้อมความพร้อมใช้งานสูง (High Availability) ช่วยให้ GISTDA ดำเนินพันธกิจด้านเทคโนโลยีอวกาศได้อย่างต่อเนื่อง
